# Operation: Doctor — Issue Flow Rules

## Active Rule
Do not open a new milestone issue before the current milestone is completed and frozen.

## Bot Response Rule
If a department bot does not answer, the assistant must not answer instead of the bot. The issue must be closed and a new issue must be opened with the correct department label.

## Label Rule
Before opening any issue, the department label must be checked.

## Single Active Milestone Rule
Only one milestone review/freeze chain should be active at a time.
